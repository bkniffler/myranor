# Fachkräfte — Charaktertabelle (Soll)

Status: canonical (Soll)

Diese Tabelle wird nach der Anwerbetabelle verwendet: `docs/rules/soll/tables/fachkraefte-anwerbetabelle.md`.

## Anwendung (Soll)

- Würfle `1w20` und wende die Eigenschaft an.
- Eigenschaften geben i.d.R. einen Bonus und eine Kehrseite.
- Wenn ein Effekt unklar ist, gilt die konservative Interpretation „nur für den Posten/den Kontext, an den die Fachkraft gebunden ist“.
- Loyalitäts‑Modifikatoren werden auf die Regel‑Skala geklemmt (`LO 0–6`).
