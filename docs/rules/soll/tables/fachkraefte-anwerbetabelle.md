# Fachkräfte — Anwerbetabelle (Soll)

Status: canonical (Soll)

Diese Tabelle wird nach einem erfolgreichen **Anwerben‑Check** verwendet (siehe `docs/rules/soll/facilities.md` → „Fachkräfte anwerben“).

## Anwendung (Soll)

1. Fachkrafttyp + Erfahrungsstufe wählen (einfach/erfahren/meisterlich) und Basis‑Kosten bezahlen.
2. Anwerben‑Check bestehen.
3. Danach: **2w6** auf dieser Tabelle.
4. Ergebnis anwenden:
   - Kostenmodifikator ist **zusätzlich** zu den Basis‑Kosten.
   - Erfahrungsstufe „wie gewünscht“ bedeutet: bleibt bei der zuvor gewählten Stufe.
   - Loyalität setzt den Start‑LO der Fachkraft (Skala: `LO 0–6`, Cap `6`, siehe `docs/rules/soll/facilities.md`).
5. Danach: `1w20` auf der Charaktertabelle: `docs/rules/soll/tables/fachkraefte-charaktertabelle.md`
