# Tabellen

Diese Dateien enthalten die **Listen/Tables** als editierbare, stabile Referenzen:

- `docs/rules/tables/events.md`: Ereignisse (Roll → Name → Effekte)
- `docs/rules/tables/market.md`: Markttabellen (Roll → Kategorie → Modifikatoren)
- `docs/rules/tables/materials.md`: Materialkatalog (Material-ID → Gruppen/Tags)

Konvention: Tabellen sind „flach“ (keine verschachtelten Effekte in einer Zelle). Details stehen unterhalb als Bullet-Listen.
