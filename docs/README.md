# Dokumentation

Diese Repo-Dokumentation ist in fünf Bereiche aufgeteilt:

- `docs/rules/`: **Regeln & Listen** (canonical für das Spielsystem; Änderungen hier sind die Basis für Code-Anpassungen)
- `docs/design/`: Architektur/Design-Entscheidungen
- `docs/dev/`: Entwickler-Dokumente (Playtest, Implementierungsstatus, Prozesse)
- `docs/reference/`: Referenzen/Quellen (z.B. Original-Regeltext)
- `docs/llm/`: LLM-spezifische Dokumente (**derived** aus `docs/rules/`)

Startpunkt für die Regeln: `docs/rules/README.md`.
